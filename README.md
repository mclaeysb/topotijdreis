# Topotijdreis.be

Explore historical maps and imagery in Belgium - Inspired by [Topotijdreis.nl](https://topotijdreis.nl) and [historical-map](https://github.com/osmbe/historical-map).

## Developing

To start a development server:

```sh
pnpm run dev
```

## Building

To build:

```sh
pnpm run build
```
